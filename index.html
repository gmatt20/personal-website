About Me


Hi! I’m Matthew, a third-year psychology student at UC Riverside, passionate about technology and software development. After discovering my love for programming in an introductory C++ class, I shifted my focus toward a career in software engineering, where I can combine problem-solving with user-centered design.

I’m currently learning web development through The Odin Project, building a solid foundation in front-end and back-end skills. My focus includes mastering HTML, CSS, and JavaScript, with plans to dive into React and back-end technologies like Node.js, Express.js, and PostgreSQL. My goal is to become a full-stack software engineer, adept at all stages of development.





My Skills


HTML


  • Semantic HTML: I use semantic tags like to improve the structure and accessibility of my web pages, ensuring a clean, understandable codebase and enhancing SEO.

  • SEO Fundamentals: Familiar with tags and attributes that enhance search engine optimization, such as meta tags, alt text for images, and structured data.

  • Responsive Layouts: I structure pages for mobile responsiveness, using techniques like flexible containers and appropriate viewport settings to ensure compatibility across devices.

  • Accessible Markup: Implement practices to support accessibility, such as alt tags for images, proper heading structure, ARIA attributes, and tab order for better usability.



CSS


  • Responsive Design and Media Queries: Proficient in designing adaptable layouts using media queries, fluid grids, and flexible images to ensure that websites look great on all screen sizes.
